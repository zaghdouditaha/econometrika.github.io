Econometrica

Econometrika is an econometric software developed entirely in C #. Based primarily on the statistical and mathematical library TahaStatMatrix (not free) i developed earlier, inspired from JAMA (java), which provides us with the main basic functions such as matrix algebra, distributions ... ect. I developed an open-source statistical library TMatrix, free published on CodePlex.

Features

Econometrika is intended primarily for Tunisians Economics students and the rest of the world. Indeed Econometrika helps students/users to do:

  • Descriptive statistics
  • Multiple Regression
  • White heteroscedasticity test, ADF stationarity test...
  • Logit and Probit model
  • Panel Data: Fixed /Random effect , Husman Test
  • Instrumental Panel Data
  • ROC curve
